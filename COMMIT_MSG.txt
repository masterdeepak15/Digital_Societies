Backend — DigitalSocieties.Social (new module):
- Social.csproj: EF Core 8, MediatR, Npgsql, FluentValidation
- Domain:
  - PostCategory constants (general/lost_found/help_wanted/for_sale/
    recommendation/warning/event/poll/emergency); AdminOnly + AutoExpiring sets
  - SocialPost aggregate: Create factory, AddImage, Pin/Unpin,
    LockComments/Unlock, SoftDelete; auto-expire for HelpWanted (24h)
  - PostComment: one-level deep threading; SoftDelete
  - PostReaction: helpful/thanks/thumbsup; upsert semantics; no dislike
  - SocialGroup + GroupMember: auto_wing/auto_floor/manual types
  - SocialPoll + PollVote + PollOption: CastVote with closed-poll guard;
    HasEnded(); AllowMultiple support
  - MarketplaceListing: pricePaise (null=free), condition, MarkSold
  - DirectoryEntry: per-field opt-in (phone/email); AdminHide/Unhide
  - PostReport: pending/reviewed/dismissed/actioned status
  - SocialEvents domain events: PostCreated, PostDeleted, EmergencyPost,
    CommentAdded, PostReacted, PollEnded, PostReported
- Application commands:
  - CreatePostCommand: enforces AdminOnly categories, creates Poll/Listing
    children, fires SignalR (feed/group/emergency channels)
  - ReactToPostCommand + RemoveReactionCommand: upsert/remove reactions
  - CommentOnPostCommand + DeleteCommentCommand: lock/expire guard,
    one-level nesting enforced
  - DeletePostCommand: owner or admin soft-delete
  - PinPostCommand, LockPostCommentsCommand: admin moderation
  - ReportPostCommand: feeds admin moderation queue
  - VotePollCommand: upsert vote, closed-poll guard
  - UpsertDirectoryEntryCommand: create-or-update own entry
- Application queries:
  - GetFeedQuery: pinned-first, newest, filtered by group/category,
    marketplace inline join
  - GetPostDetailQuery: full post + threaded comments + grouped reactions
    + poll results with per-option percentages + marketplace detail
  - GetDirectoryQuery: opted-in only, hidden entries excluded, ILike search
- Infrastructure:
  - ISocialHubNotifier (ISP interface) + NullSocialHubNotifier null-object
  - SocialDbContext: "social" schema, all 10 tables, JSONB for image_urls/
    poll options/vote option_ids, unique indexes on reactions + group_members
    + poll_votes + directory, global soft-delete query filter on posts
  - SocialServiceExtensions.AddSocialModule()
